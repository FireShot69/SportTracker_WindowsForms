README SPORT TRACKER WINFORMS

Описание:
Простое приложение на C# Windows Forms для учета спортсменов и тренировок.

Возможности:
- Добавление спортсмена (имя и вид спорта)
- Добавление тренировки (длительность и заметки)
- Удаление спортсменов
- Удаление тренировок
- Просмотр списка спортсменов
- Просмотр списка тренировок
- Просмотр статистики

Как пользоваться:
1. Запустить программу через Visual Studio
2. Ввести имя спортсмена и вид спорта
3. Нажать кнопку "Добавить спортсмена"
4. Выбрать спортсмена из списка
5. Добавить тренировку
6. При необходимости удалить записи

Технологии:
- C#
- Windows Forms
- ООП
- Списки (List)

Структура проекта:
Form1.cs - логика программы
Program.cs - запуск приложения
Program.Designer.cs - интерфейс
Models - классы данных (Athlete, Training)
Services - бизнес-логика (SportService)

Особенности:
- Данные хранятся в памяти
- После закрытия программы данные удаляются
- Проект учебный

Статус:
Приложение работает и готово к использованию
